# FE-02 — Conversation Thread Rendering

## Story
As a user, I want my questions and the assistant's answers to appear as a vertical conversation thread so that I can follow the exchange.

## Context
Submitted questions and returned answers appear as a vertical conversation thread (PRD FR-2).

## Acceptance Criteria
- [ ] User message and assistant answer render in a vertical thread (FR-2).
- [ ] Clear visual distinction between user and assistant turns.
- [ ] Thread auto-scrolls to the latest message.
- [ ] Layout holds up for long conversations.
